learning basics of git


git installation and setup

| Task | Command | | ----------------- | -------------------------------------------------- | | Check Git version | `git --version` | | Set username | `git config --global user.name "Shivakumar"` | | Set email | `git config --global user.email "you@example.com"` |



repo initialization

| Task | Command | | -------------------- | ------------------------------------ | | Create folder | `mkdir project-name` | | Move into folder | `cd project-name` | | Start Git tracking | `git init` | | Check if it's a repo | `git status` or `ls -a` (see `.git`) |



Tracking Changes

| Step | Command | Purpose | | ------------------- | ------------------------- | -------------------- | | Check file status | `git status` | See what's changed | | Stage a file | `git add filename` | Move file to staging | | Stage all files | `git add .` | Stage everything | | Commit changes | `git commit -m "message"` | Save changes | | View commit history | `git log` | Show commits made |



Undoing Mistakes

| Situation | Command | Description | | ------------------------------- | --------------------------- | -------------------------- | | Undo file changes (unstaged) | `git restore file` | Revert file to last commit | | Unstage a file | `git restore --staged file` | Remove from staging | | Undo last commit (keep changes) | `git reset --soft HEAD~1` | Uncommit changes | | Undo commit and unstage | `git reset HEAD~1` | Uncommit and unstage | | Delete all changes (dangerous) | `git reset --hard HEAD` | Discard everything ðŸ˜± |



Branching

| Task | Command | Purpose | | ------------------------- | ------------------------------------------------------------ | ---------------------- | | List branches | `git branch` | Shows local branches | | Create + switch branch | `git checkout -b branch-name` or `git switch -c branch-name` | New branch and switch | | Switch branches | `git switch branch-name` | Go to another branch | | Merge into current branch | `git merge branch-name` | Combine branch changes | | Delete a branch | `git branch -d branch-name` | Clean up after merge |
